## Simple War
Match Plugin for War servers CS/CZ<br/>

### Console Command
amx_startmatch<br/>
amx_stopmatch<br/>
amx_pausematch<br/>
amx_unpausematch<br/>

### Chat Command
/start<br/>
/stop<br/>
/pause<br/>
/unpause<br/>

### Installations
> Put .amxx in amxmodx/plugins/<br/>

### Requirements:
AMX 1.9.0+ Dev or ReAMX<br/>
ReAPI<br/>
ReHLDS & ReGameDLL<br/>
